# クラスとスタイル

## この演習で学習する内容
- HTMLのclassのバインディング
- HTMLのstyleのバインディング

## 演習内容
[最初に作成](./hello-world.md)した`Hello World!`を表示するアプリケーションに、`<p>`タグに以下のクラスとスタイルを適用して見た目を変えてみましょう。

- class: `message`
- style: `color: white; background-color: red;`
